🧬 The Realm of Adrisha — Interactive Portfolio

An immersive game-inspired developer portfolio built with modern web technologies.
Instead of a traditional webpage, this project presents my work through a cinematic tactical interface and interactive battle experience.

Users navigate through different sections of the portfolio by interacting with avatars inside a battlefield, unlocking information about my work, skills, and collaborations.

🎮 Experience Overview

The portfolio is structured as a multi-mode interactive environment:

1️⃣ Character Selection

Users begin by selecting a role to enter the system.

2️⃣ Mode Selection

Two primary exploration modes are available:

Tactical View – structured exploration of portfolio sections

Combat Mode – interactive battle interface to unlock information

3️⃣ Tactical Interface

The tactical system organizes the portfolio into intelligence modules:

Profile — about me

Core Abilities — skills & technologies

Mission Log — projects and work

Alliances — collaborations & communities

Contact Channels — ways to reach me

4️⃣ Battle Mode

A gamified experience where users:

Aim and fire a laser weapon

Hit interactive avatars

Unlock portfolio sections through gameplay

Gain progress via a score system

5️⃣ Victory Screen

After discovering all avatars, the user unlocks the Victory screen, confirming completion of the realm.

🎯 Design Philosophy

This project explores the idea that a portfolio should be an experience, not just a webpage.

Instead of listing information statically, users discover it through interaction, turning the portfolio into a small interactive world.

The goal was to blend:

game design principles

modern web technologies

cinematic UI storytelling
